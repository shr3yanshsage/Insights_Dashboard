A modern, interactive data analytics dashboard built with **React**, **TailwindCSS**, and **Chart.js** (via `react-chartjs-2`).  
This dashboard visualizes business data with dynamic and responsive charts to help explore meaningful insights.

Overview:
The **Insights Dashboard** displays interactive analytics based on customer and product data including:

- Product distribution
- Average product age
- API call usage
- Customers per state
- Company size vs API usage

Built as a clean, professional Single Page Application (SPA) suited for FinTech / SaaS analytics.
